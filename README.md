# My Website

A simple HTML-based website project.

## Project Structure

```
├── index.html        - Main page
├── css/
│   └── styles.css   - Main stylesheet
├── js/
│   └── script.js    - Main JavaScript file
└── images/          - Store all images and graphics here
```

## Getting Started

1. Open `index.html` in a web browser
2. Add your content to the HTML files
3. Style with CSS in `css/styles.css`
4. Add interactivity with `js/script.js`
5. Place all images in the `images/` folder

## File Organization

- **HTML Files**: Main content structure
- **CSS**: Styling and layout
- **JavaScript**: Interactivity and functionality
- **Images**: All images, icons, and graphics

Feel free to expand this structure as your project grows!
